# AI Novel Plot Generator v2.10

A structured, logic-driven novel generation system with:
- Literary masters techniques (Jin Yong, Liu Cixin, Gu Long, Murakami, Yu Hua)
- Web novel authors techniques (I Eat Tomatoes, Er Gen, Mao Ni, Zhai Zhu, Hui Shuohua de Zhouzi, Jiang Nan)
- Web novel reading sense (male & female frequency)
- Anti-trope blending (anti-融條) mechanisms for female frequency

## Features
- Multi-round logical deduction
- Character agents simulation
- Theme and emotional depth tracking
- Long-term memory
- Strong chapter hooks and pacing for web novels

This system is designed for Tomato Novel (番茄小说) style short and long stories, both male and female frequency.
