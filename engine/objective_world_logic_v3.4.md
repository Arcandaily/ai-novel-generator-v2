# 客观世界逻辑驱动系统 v3.4

## 核心理念
除了人物内在逻辑驱动剧情外，**世界本身也存在客观运行规律**，这些规律会独立或半独立地产生事件、后果和连锁反应，从而推动剧情发展。

这让故事更具真实感和不可预测性，避免完全依赖人物主观意志。

## 1. 客观世界逻辑的类型

### 1.1 世界基础规则（硬规则）
- 物理法则、社会运行规则、权力结构、资源分配逻辑等。
- 例如：在一个严格的等级社会中，低阶层人物的上升必然会触动既得利益者。

### 1.2 历史与政治惯性
- 世界正在发生或即将发生的宏观事件（战争、政策变化、权力更迭、经济危机等）。
- 这些事件有自己的发展逻辑，不完全受单一人物控制。

### 1.3 蝴蝶效应与连锁反应
- 之前发生的事件会在时间线上产生延迟的、间接的后果。

### 1.4 随机性与不可控因素
- 世界中存在一定程度的随机事件（意外、运气、自然灾害、信息误传等）。
- 这些事件可以打破人物的计划，制造新的冲突或转机。

## 2. 客观逻辑如何推动剧情

### 方式一：独立生成事件
世界模型可以根据当前状态，主动生成“客观事件”，然后让人物去应对。

### 方式二：对人物行为的反馈
人物做出某个行为后，世界会根据客观规则产生**必然或高概率的后果**。

### 方式三：时间线自然推进
即使人物什么都不做，时间也会流逝，世界会按照自己的逻辑发展。

## 3. 客观逻辑 + 角色逻辑的结合机制

推荐采用**双驱动模式**：
1. **角色驱动**（主动）：人物根据自己的欲望、恐惧、习惯主动做出选择。
2. **客观驱动**（被动/半被动）：世界根据客观规则产生事件或后果，人物必须应对。

## 4. 在 6W1H 中的体现

在进行 6W1H 扩展时，必须同时考虑客观逻辑：
- **When**：不仅考虑人物想什么时候行动，还要考虑世界当前的状态。
- **Why**：不仅考虑人物动机，还要考虑客观环境是否“逼迫”或“允许”。
- **How**：人物的执行方式必须符合世界规则。
