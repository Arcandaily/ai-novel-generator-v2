# Self-Critique 详细检查清单 v3.13

## 使用说明
本清单用于 Narrator 生成初稿后，由 Self-Critique 模块进行系统性审查。每个检查项需给出 **通过 / 轻微问题 / 严重问题** 判定，并附简要说明。严重问题必须触发 Rewriting 或拒绝。

---

## 第一层：基础逻辑一致性（必须全部通过）

### 1.1 人设一致性
- [ ] 人物行为是否符合其性格、习惯、思维方式和底线？
- [ ] 人物的动机是否合理？是否与其核心欲望/恐惧一致？
- [ ] 人物在当前情境下的反应是否自然？

### 1.2 因果逻辑一致性
- [ ] 事件的起因是否充分？
- [ ] 事件的后果是否合理？是否产生了可追踪的连锁反应？

### 1.3 时间线一致性
- [ ] 事件发生的时间是否合理？
- [ ] 信息传递是否符合世界当前科技/社会水平？
- [ ] 是否存在时间bug？

### 1.4 世界模型一致性
- [ ] 事件是否符合世界基础规则？
- [ ] 脑洞使用是否支付了合理代价？

---

## 第二层：题材与风格一致性

### 2.1 题材核心一致性
- [ ] 当前事件是否主要服务于核心题材？
- [ ] 是否引入了与题材严重冲突的元素？

### 2.2 风格与阅读感一致性
- [ ] 节奏是否符合当前题材要求？
- [ ] 是否有合适的章末钩子？

---

## 第三层：主题与情感深度

### 3.1 主题服务度
- [ ] 当前节点是否对核心主题有推进？

### 3.2 情感冲击力
- [ ] 是否有明显的情感起伏？
- [ ] 虐与甜/成长的节奏是否合理？

---

## 第四层：角色驱动与客观逻辑

### 4.1 角色驱动一致性
- [ ] 人物的行为是否主要由其内在逻辑驱动？

### 4.2 客观世界逻辑
- [ ] 是否有客观世界规则或事件在推动剧情？

---

## 第五层：反融梗与原创性

### 5.1 融梗风险评估
- [ ] 当前情节、人物关系、桥段是否高度相似其他爆款作品？
- [ ] 是否有明显的“抄作业”痕迹？

---

## 第六层：整体质量与网文阅读感

### 6.1 整体节奏与沉浸感
- [ ] 是否有明显的阅读粘性？
- [ ] 感官描写、心理描写是否平衡？

### 6.2 整体评价
- **通过**：可直接输出
- **轻微问题**：小幅修改后输出
- **严重问题**：必须重写
- **拒绝**：逻辑或题材问题过大，建议回滚或重新推演

---

## 使用流程建议

1. Narrator 生成初稿后，Self-Critique 按以上六层逐项检查。
2. 每项给出判定 + 简要说明。
3. 汇总后决定是直接输出、轻改、重写还是拒绝。
4. 所有审查记录由 Archivist 保存。
