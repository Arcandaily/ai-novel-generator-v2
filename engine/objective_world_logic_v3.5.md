# 客观世界逻辑驱动系统 v3.5（实用强化版）

## 核心理念
世界本身拥有**客观运行规律**，这些规律会独立或半独立地产生事件、后果和变化，从而推动剧情发展，使故事更具真实感、不可预测性和逻辑自洽性。

## 1. 客观世界逻辑的主要类型（细化）

### 1.1 硬性世界规则
- 物理/超自然法则、社会运行规则、权力结构、资源稀缺性等。
- 示例：在一个严格的门阀社会中，底层人物的快速崛起必然会遭到既得利益集团的联合打压。

### 1.2 宏观历史与政治进程
- 世界正在进行或即将发生的重大事件（王朝更迭、战争、政策改革、经济危机等）。
- 这些事件有自己的发展轨迹和时间表。

### 1.3 蝴蝶效应与延迟连锁反应
- 之前的行为或事件会在未来产生间接、延迟的后果。

### 1.4 系统性随机性与意外
- 世界中存在一定概率的随机事件（意外、运气、自然灾害、信息误传等）。
- 这些事件可以打破人物的计划。

### 1.5 资源与环境动态
- 资源枯竭、新资源出现、环境变化、流行病等。
- 会迫使人物改变策略或卷入新冲突。

## 2. 客观逻辑生成事件的机制

### 机制A：周期性/阶段性事件
世界模型可以设定一些“定时”或“阶段性”的大事件。

### 机制B：阈值触发事件
当某个条件达到阈值时，自动触发客观事件。

### 机制C：人物行为诱发客观反应
人物的某个行为会触发世界的“自动反馈系统”。

### 机制D：纯客观随机事件
完全独立于当前主要人物行动的随机事件，用于增加不可预测性。

## 3. 客观事件与人物反应的互动

客观事件发生后，人物会根据自己的人设做出反应。不同人物对同一客观事件的反应差异很大。

推荐结构：客观事件（世界推动） → 人物必须应对（人物驱动） → 产生新后果（再次进入世界逻辑）。

## 4. 在 6W1H 中的体现

进行 6W1H 扩展时，必须同时考虑客观逻辑：
- **When**：不仅考虑人物想什么时候行动，还要看世界当前处于什么阶段。
- **Why**：不仅考虑人物动机，还要考虑客观环境是否“逼迫”或“允许”。
- **How**：人物的执行方式必须符合世界当前规则。

## 5. 实际应用建议

1. World Model 需要维护“宏观状态面板”。
2. Logical Deduction Engine 在推演时，应同时生成“人物主动分支”和“客观事件分支”。
3. 重要转折点建议至少有一个客观因素参与。
4. 反融梗：客观逻辑驱动的事件通常比纯人物主观行为更不容易被判定为融梗。
