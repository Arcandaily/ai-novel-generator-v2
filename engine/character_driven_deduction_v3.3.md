# 角色驱动演绎系统 v3.3（强化实用版）

## 核心理念
**以完整人设为中心**，通过人物的内在逻辑（动机、欲望、弱点、习惯、应对措施）来自然推动剧情发展。

## 1. 完整人设档案模板（v3.3 增强版）

### 1.1 基础身份层
- 姓名、年龄、性别、显著外貌特征
- 社会身份与当前处境
- 公开形象 vs 真实形象（重要，用于马甲文或反差）

### 1.2 内在驱动层（最重要）
- **核心欲望**（最想得到什么？短期 vs 长期）
- **核心恐惧**（最害怕什么？最不愿意面对的事）
- **价值观与底线**（什么可以做，什么绝对不能做？）
- **情感执念/创伤**（过去哪件事对现在影响最大？）

### 1.3 行为与决策层
- **习惯性反应模式**（紧张时会做什么？生气时如何表现？面对诱惑的默认反应）
- **思维偏好**（理性计算型 / 情绪驱动型 / 直觉型 / 长期规划型）
- **沟通与表达风格**（是否善于隐藏情绪？习惯用什么方式说服别人？）
- **风险偏好**（冒险型 / 保守型 / 机会主义型）

### 1.4 能力与资源层
- 擅长领域与明显弱项
- 可用资源（人脉、财富、信息、特殊能力、身体条件）
- 当前限制条件（身份束缚、外部压力、身体/心理状态）

### 1.5 关系动态层
- 与其他关键人物的关系性质（盟友/敌人/暧昧/利用/依赖等）
- 关系中的权力结构（谁更主动？谁更需要对方？）
- 关系中的情感状态（信任度、敌意度、依赖度）

## 2. 角色驱动的 6W1H 扩展实操

### 核心原则
当需要推进剧情时，**优先从主要人物的内在状态出发**，使用 6W1H 进行扩展，而不是先想事件再套人物。

### 示例：女主得知绿茶在背后说了坏话

**当前人物状态简要**：
- 女主：最近刚经历事业挫折，自尊心受损，对绿茶一直有戒心，核心欲望是“被认可”和“掌控局面”，弱点是“容易被针对自尊心的事激怒”。
- 绿茶：爱面子，实际在男主面前没有安全感，习惯用小手段维持地位。

**以女主为中心进行 6W1H 扩展**：

**Who**：女主是直接找绿茶？还是找男主先确认？还是找其他盟友商量？
**When**：立刻行动（情绪冲动）？还是冷静下来后行动？
**What**：她准备怎么回应？是直接反击、冷暴力、还是先收集证据？
**Where**：她选择在什么场合回应？公开场合、私下、还是通过第三方？
**Why**：她为什么这么做？是出于愤怒？自尊？想保护自己的形象？还是想借机试探男主的态度？
**How**：她具体怎么执行？语气、姿态、用什么话术？

通过以上扩展，系统可以生成多个合理但风格不同的发展方向。

### 多角色并行扩展
同时从绿茶、男主等关键人物角度进行扩展，观察不同人物的反应如何相互影响，最终形成更立体的剧情节点。

## 3. 动态人设更新机制

人物不是静态的。每次重要事件后，必须更新人物档案中的情感状态、关系信任度、新资源或限制等。

## 4. 与多维度控制的结合

在 6W1H 扩展时，必须同时检查人设一致性、世界模型、时间线、因果逻辑、主题、情感冲击力、网文节奏、融梗风险。

## 5. 实施建议

1. 重要阶段必须更新人设档案
2. 优先从主角 + 1-2个最相关人物开始 6W1H 扩展
3. 扩展时优先考虑人物的“弱点”和“习惯”
4. 保留多个合理方向供选择
5. 每次扩展后立即进行多维度检查
