# 我吃西红柿写作技法蒸馏 v3.24（引擎集成版）

## 核心风格定位
稳健的体系流 + 主角长期成长 + 世界观宏大但逻辑清晰 + 节奏平稳有力，适合长篇连载。

## 1. 与角色驱动的结合

**人设特点**：
- 主角通常有明确、长期的目标。
- 性格坚韧、正面、有底线，会经历挫折和成长。
- 强调“努力 + 机缘 + 智慧”的结合。

**在 6W1H 中的应用**：
- 成长节奏相对稳健，重要突破有铺垫和代价。
- 主角的动机清晰且持久。

**冲突解决倾向**：
- 优先保证人设一致性。
- 当与世界逻辑冲突时，通常通过“主角努力 + 合理代价”来解决。

## 2. 与客观世界逻辑的结合

**世界观特点**：
- 世界宏大但规则清晰。
- 强调“天道”或世界规则的客观性。

**在引擎中的应用**：
- 客观事件应有自己的时间节奏和触发条件。
- 主角的行为会合理地触发世界的反馈。
- 蝴蝶效应和连锁反应应被记录和利用。

## 3. 与题材一致性的结合

**典型题材**：玄幻、仙侠、科幻、星际等体系流。
- 事件生成应主要围绕修炼、势力、资源、命运等核心元素。

## 4. 与时间线推演的结合

**节奏特点**：成长节奏相对稳健。
- 适合使用时间跳跃来展示长期成长，但跳跃后需合理更新状态。

## 5. 与 Self-Critique 的结合

**重点审查维度**：
- 人设一致性、因果逻辑、世界模型、题材一致性、反融梗。

**常见问题提示**：
- 主角成长过快或过于顺利
- 世界规则前后不一致
- 重要事件缺乏铺垫和因果

## 6. 与主题和情感深度的结合

**常见主题**：个人成长与命运抗争、守护与责任、探索未知与突破极限。

## 7. 实施建议（引擎使用时）

- 生成分支时，优先保留“主角通过努力+合理机缘获得成长”的方向。
- 重要节点建议同时考虑主角主动推动和客观世界自然发展。
- Self-Critique 时重点检查成长节奏是否合理、因果是否清晰、世界规则是否一致。
